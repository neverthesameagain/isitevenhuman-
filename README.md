# isitevenHuman — AI Text Detector
Repository: https://github.com/neverthesameagain/isitevenhuman
A **114-feature stylometric AI text detection engine** built on NLP, deep learning embeddings, and classical statistical analysis. Paste any text and instantly see whether it was written by a human or generated by AI — with per-sentence color-coded breakdowns and full feature transparency.

## Features

- **114 Stylometric Dimensions** — Lexical diversity, Zipf's law deviation, POS tagging, Shannon entropy, discourse markers, SBERT coherence, and more
- **Per-Sentence Highlighting** — Each sentence is independently classified and color-coded (🔴 AI / 🟢 Human)
- **Feature Group Toggles** — Select which analysis categories to include in the prediction (9 toggleable groups)
- **Model Hot-Swap** — Switch between `New` and `Old` XGBoost model weights at runtime
- **Full Feature Transparency** — Every extracted metric is displayed with searchable, filterable cards
- **Beautiful Dark UI** — Glassmorphism design with animated gauges and micro-interactions

## Quick Start (Local)

```bash
# 1. Clone the repository
git clone https://github.com/neverthesameagain/isitevenhuman.git
cd isitevenhuman

# 2. Create virtual environment
python3 -m venv .venv
source .venv/bin/activate

# 3. Install dependencies
pip install -r requirements.txt
python -m spacy download en_core_web_sm

# 4. Run
chmod +x run.sh
./run.sh
```

Open **http://127.0.0.1:8000** in your browser.

## Feature Groups

| Group | Features | Description |
|-------|----------|-------------|
| Sentence Structure | 7 | Length stats, burstiness, CV |
| Lexical Diversity | 14 | TTR variants, hapax ratio, vocab richness |
| Discourse Markers | 14 | Template phrases, safe vocab patterns |
| Entropy & Repetition | 17 | Shannon entropy, n-gram reuse rates |
| Zipf's Law | 18 | Frequency distribution shape analysis |
| POS Tagging | 18 | Part-of-speech ratios, syntactic transitions |
| Informality | 9 | Typos, contractions, pronoun usage |
| Structural Layout | 16 | Paragraph stats, list detection, headers |
| SBERT Coherence | 6 | Sentence embedding similarity flow |

## API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `GET`  | `/` | Serve the web UI |
| `GET`  | `/api/models` | List available model variants |
| `GET`  | `/api/feature-groups` | List toggleable feature groups |
| `POST` | `/api/predict` | Run full analysis on input text |

### POST `/api/predict` — Request Body

```json
{
  "text": "Your text here...",
  "model_variant": "new",
  "enabled_groups": ["sentence", "lexical", "entropy", "sbert"]
}
```
